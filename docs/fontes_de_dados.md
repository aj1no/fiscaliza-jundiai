# Fontes de Dados - Fiscaliza Jundiaí

O sistema coleta dados de três fontes principais do município de Jundiaí/SP.

## 1. Imprensa Oficial de Jundiaí
- **URL**: [https://imprensaoficial.jundiai.sp.gov.br/](https://imprensaoficial.jundiai.sp.gov.br/)
- **Dados Coletados**: Edições do diário oficial, decretos, portarias, editais de licitação e atos administrativos.
- **Formato**: Geralmente PDF e listagem em HTML.

## 2. Câmara Municipal de Jundiaí (Resumos das Sessões)
- **URL**: [https://jundiai.sp.leg.br/atividade-legislativa/resumos-das-sessoes](https://jundiai.sp.leg.br/atividade-legislativa/resumos-das-sessoes)
- **Dados Coletados**: Resumos de sessões ordinárias e extraordinárias, votações, proposições (projetos de lei, indicações, moções).
- **Formato**: HTML e anexos via SAPL.

## 3. Portal da Transparência (Prefeitura de Jundiaí)
- **URL**: [https://transparencia.jundiai.sp.gov.br/](https://transparencia.jundiai.sp.gov.br/)
- **Dados Coletados**: Receitas, despesas, contratos, licitações, empenhos, pagamentos, fornecedores e órgãos públicos.
- **Formato**: HTML, CSV e PDF.

## Política de Coleta
- Periodicidade: Diária ou conforme publicação.
- Ética: Respeito à LGPD e anonimização de dados pessoais sensíveis.
- Rastreabilidade: Cada registro mantém o link para a fonte original.
